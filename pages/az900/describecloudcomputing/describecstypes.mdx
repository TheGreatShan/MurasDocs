# Cloud service types

!["model"]("https://learn.microsoft.com/en-us/training/wwl-azure/describe-cloud-service-types/media/shared-responsibility-b3829bfe.svg")

## IaaS (Infrastructure as a service)
### What is IaaS?
With IaaS I got the maximum amount of control for my cloud resources. Everything related to infrastructure is done by Azure. in the model above you see what is done my Microsoft. 
## Where does it make sense?
- Lift and shift migration: I can move my existing infrastructure to the cloud.

## PaaS (Platform as a service)
### What is PaaS?
In this case, Azure is responsible for nearly everything. I just need to care about my application. All licenses will be bought or rented by Microsoft.
## Where does it make sense?
- Development: I can develop my application without caring about the infrastructure.

# SaaS (Software as a service)
### What is SaaS?
In this case, Azure is responsible for everything. I just install a software by a third party. I don't need to care about anything.

## Where does it make sense?
- Microsoft 365 (Office 365)
- Email services
- Finance tracker
- ...
